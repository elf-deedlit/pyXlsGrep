# pyXlsGrep
Pythonでexcelファイルをgrep

# 使い方

```
usage: pyXlsGrep.py [-h] [--basepath BASEPATH] findstr

positional arguments:
  findstr              検索文字列

optional arguments:
  -h, --help           show this help message and exit
  --basepath BASEPATH  検索パス
```

# モジュール
openpyxlが必要です。

```Bash
$ pip install openpyxl
```
